Project Name

Lesson Matching Platform (Zoom-based)

1. Overview

Build a web-based MVP prototype for a platform that connects Learners and Teachers for online lessons.

Lessons will be conducted using Zoom meetings.

The system should allow:

user authentication

teacher profile listing

learner search

lesson booking

Zoom meeting link generation

chat messaging

lesson completion and review

This is an MVP prototype, so prioritize simplicity and usability.

2. Tech Stack

Use the following stack:

Frontend

Next.js

Tailwind CSS

Backend

Node.js

Express

Database

PostgreSQL

Auth

JWT authentication

Video

Zoom Meeting API

Chat

WebSocket (simple real-time chat)

Deployment ready but not required.

3. User Roles

Two roles exist in the system.

Learner

A user who books lessons.

Teacher

A user who provides lessons.

Users choose their role during registration.
